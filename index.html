If you add parameters end of URL, you can use these in a javascript object with urlparam()

example 1

In this https://sinseiki.github.io/urlparam/index.html?a=1&b=abc 

        const params = urlparam();

you can use parameters in 'params' object like

        const params = {a:'1', b:'abc'}

example 2

In this https://sinseiki.github.io/urlparam/index.html?tabNum=2

you can use tabmenu colored on the second tab with this code.

        const params = urlparam();
        const lis = document.querySelectorAll('li');
        if(params.tabNum) lis[params.tabNum - 1].classList.add('on');
